## python-mastery
Structured Python exercises from foundation to AI-ready.

## Structure

```
python-mastery/
├── level-1-foundation/
│   ├── | Comprehensions, strings, functions, files, exceptions 
│
├── level-2-intermediate/
│   ├── | OOP, decorators, generators, type hints 
│
├── level-3-advanced/
│   ├── | Async, testing, profiling, modules  
│
├── level-4-python-for-ai/
│   ├── | NumPy, Pandas, ML pipelines, data cleaning 
│   ├── | APIs, file parsers, vector stores 

```

## Levels

| Level | Focus | Duration |
|---|---|---|
| 1 — Foundation | Comprehensions, strings, functions, files, exceptions | 2–3 weeks |
| 2 — Intermediate | OOP, decorators, generators, type hints | 3–4 weeks |
| 3 — Advanced | Async, testing, profiling, modules | 3–4 weeks |
| 4 — Python for AI | NumPy, Pandas, ML pipelines, data cleaning | 3–4 weeks |
| Bonus | APIs, file parsers, vector stores | ongoing |

## Goal

Complete all 4 levels → build the 4 projects → move on to ML fundamentals.
